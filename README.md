## 📖 專案概述 (Overview)

**呼嚕嚕小鎮** 是一個基於 Web 的單頁應用程式 (SPA)，旨在將繁瑣的班級經營轉化為有趣的村民互動。不同於傳統生硬的計分系統，我們引入了「自動化獎懲」、「三級貨幣」、「職務發薪」、「道具商店」與「財富排行榜」等機制，讓學生在遊戲化的經濟體驗中學習責任與自律。

## 🌟 核心功能特色 (Key Features)

我們將班級經營拆解為五大支柱：**經濟循環**、**職務責任**、**空間管理**、**智慧自動化** 與 **視覺回饋**。

### 1. 橘喵商店與道具系統 (Economy)
讓獎勵不再只是數字的堆疊，而是具有真實購買力的資產。
- **🐱 橘喵商店**：老師可自訂商店名稱與圖示，上架各種虛擬或實體獎勵卡片。
- **🎒 道具背包**：學生購買後，商品會進入個人「背包」，並扣除對應貨幣。
- **🎫 核銷機制**：採「臨櫃核銷」制，由老師操作扣除道具，並自動記錄使用歷程。
- **🏆 財富排行榜**：依學期總獲取點數排名，前五名擁有專屬徽章，支援 Excel 匯出。

### 2. 金融與職務體系 (Management)
打造班級微型經濟體，體驗真實社會運作。
- **💰 三級貨幣**：支援 **積分 (Point)**、**小魚乾 (Fish)**、**貓餅乾 (Cookie)** 三種單位換算。
- **📒 擬真存摺**：紀錄每一筆收支（薪資、罰款、消費），支援「撤銷」與「修正」功能。
- **👔 職務薪資**：支援「每週 / 每月 / 每學期」一鍵發薪，可設定班長、路隊長等職務薪資。
- **📊 累計追蹤**：系統底層持續累計學生的總獲取點數（totalEarned），不受商店消費影響。

### 3. 座位表與教室空間 (Spatial)
用拖曳打造專屬教室佈局。
- **🪑 拖曳式座位表**：全螢幕編輯器，CSS Grid 自由調整行列數（最大 12×12）。
- **🧩 教室物件系統**：內建電腦、講桌、門窗等 11 種物件 + 自訂物件功能。
- **👁️ 視角切換**：老師視角 / 學生視角一鍵切換，Grid 旋轉 180° 文字保持正向。
- **📊 匯出與列印**：支援 Excel 下載與 A4 橫式列印。

### 4. 智慧自動化 (Automation)
減少手動操作，讓系統幫你處理瑣碎事務。
- **⚡ Smart Task Logic**：任務狀態變更時自動觸發獎懲（每日任務獎勵 / 遲交扣分 / 缺交扣分）。
- **↩️ Smart Undo**：更正狀態時自動撤銷舊有的獎懲交易，保持帳目正確。
- **📝 自動免交規則**：為特定學生設定關鍵字 + 任務類型配對，新增任務時自動標記免交。
- **🏠 在家自學機制**：標記在家自學的學生後，自動排除於任務統計、小隊計算、座位表。

### 5. 沉浸式儀表板 (Dashboard)
- **三欄式佈局**：左側日曆/公告、中間任務板、右側村民廣場，資訊一目瞭然。
- **🎫 村民護照**：整合大頭像、職務徽章、資產總覽與任務狀態的個人中心。
- **✨ 30 種動物村民**：可愛的動物 Emoji 頭像（已排除豬），每班獨立隨機分配。
- **📺 投影模式**：橫向雙欄佈局——左側顯示隔日任務、右側顯示未來已排定任務。
- **☁️ 資料安全**：Local-First 架構，資料優先儲存於瀏覽器，支援 Google Drive 雲端備份與跨設備還原。

## 🎓 新手教學 (Getting Started for Teachers)

> 第一次使用呼嚕嚕小鎮？只要跟著以下四個階段，就能在 10 分鐘內完成基礎設定！

### 📌 階段一：建立你的村莊（1 分鐘）

1. 開啟網站，點選 **「+ 建立新班級」**
2. 輸入班級代號（如 `301`）與學生人數
3. 點選 **「建立」**——恭喜，你的村莊誕生了！

> 💡 每位學生會自動獲得一隻隨機動物頭像，點擊村民卡片即可進入「護照」修改姓名。

### 📌 階段二：設定經濟規則（3 分鐘）

1. 點擊頂端功能列的 **⚙️ 村莊設定**
2. 在「**貨幣設定**」確認三級貨幣名稱與匯率（預設：100 積分 = 1 小魚乾，1000 積分 = 1 貓餅乾）
3. 在「**職務設定**」建立幹部/打掃/其他職務，設定薪資與發放週期
4. 在「**行為規範**」設定加扣分項目（如：作業缺交 -200 分、發表優良 +100 分）
5. 在「**自動化設定**」調整每日任務獎勵、遲交/缺交自動扣分金額

### 📌 階段三：日常使用（每天 2 分鐘）

| 操作 | 怎麼做 |
|------|--------|
| **發布任務** | 在中間「任務板」點擊 ＋，輸入任務名稱（如「國語習作」），選擇類型後送出 |
| **標記狀態** | 在右側「村民廣場」點擊學生卡片 → 在護照中對任務標記「準時 / 遲交 / 缺交 / 請假」 |
| **快速獎懲** | 在護照的「行為」分頁，點擊預設的加扣分按鈕即可一鍵操作 |
| **查看存摺** | 在護照的「存摺」分頁，可查看完整交易紀錄與目前餘額 |

> 💡 當所有任務都標記為「準時」時，系統會自動發放每日任務獎勵！

### 📌 階段四：進階功能（按需使用）

| 功能 | 說明 |
|------|------|
| **🏆 財富榜** | 頂端功能列 → 查看全班點數排行，匯出 Excel 作為學期報告依據 |
| **🐱 橘喵商店** | 頂端功能列 → 讓學生用累積的點數兌換獎勵卡片 |
| **🪑 座位表** | 頂端功能列 → 拖曳安排座位，支援列印與 Excel 匯出 |
| **📺 投影模式** | 任務板右上角 → 全螢幕顯示明日作業，適合班級投影 |
| **☁️ 雲端備份** | 村莊設定 → 備份分頁 → 連接 Google Apps Script 進行跨設備同步 |
| **💼 發放薪資** | 村莊設定 → 職務分頁 → 一鍵為所有有職務的學生批次入帳 |

## 🚀 快速開始 (Quick Start for Developers)

### 💻 開發環境建置

1.  **Clone 專案**
    ```bash
    git clone https://github.com/NeriakTo/purr-purr-town.git
    cd purr-purr-town
    ```

2.  **安裝依賴**
    ```bash
    npm install
    ```

3.  **啟動開發伺服器**
    ```bash
    npm run dev
    ```

### ⚙️ 部署至 Github Pages

```bash
npm run deploy    # 自動 build + 部署至 GitHub Pages
```

## 📐 技術架構 (Architecture)

| 層級 | 技術 | 說明 |
|------|------|------|
| **前端框架** | React 19 | 使用最新的 React Hooks 與 Functional Components |
| **建置工具** | Vite 7 | 極速的 HMR 開發體驗 |
| **樣式系統** | Tailwind CSS 4 | CSS-first 配置，Utility-first 快速構建 UI 與 RWD |
| **拖放引擎** | @dnd-kit | 觸控友善的 DnD 框架，驅動座位表拖曳功能 |
| **圖示庫** | lucide-react | 輕量 SVG 圖示 + 原生 Emoji 頭像系統 |
| **資料儲存** | LocalStorage | JSON 格式儲存班級資料 (`ppt_cache_class_{id}`) |
| **雲端後端** | Google Apps Script | 提供 POST/GET API 進行 Google Drive 雲端備份 |
| **匯出引擎** | ExcelJS | 動態載入，支援座位表、職務表、任務狀態、財富榜 Excel 匯出 |
| **日期處理** | date-fns | 輕量日期格式化與計算 |
| **日曆元件** | react-calendar | 村莊日誌的日期導航 |
