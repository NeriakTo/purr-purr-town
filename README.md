# 🐾 呼嚕嚕小鎮 (Purr Purr Town)

**版本：v1.9.0**

一個以「動物森友會」風格為靈感的班級管理系統，提供溫馨、療癒的使用體驗。

## 📋 專案簡介

呼嚕嚕小鎮是一個專為教師設計的班級管理網頁應用程式，透過可愛的村莊主題，將班級管理變得輕鬆有趣。系統支援多班級管理、任務追蹤、學生狀態監控等功能，並提供直覺的操作介面。

## ✨ 主要功能

### 🏘️ 多班級管理
- 建立並管理多個班級（村莊）
- 為班級設定別名，增加個人化
- 快速切換不同班級

### 📅 日曆導航
- 使用日曆查看不同日期的資料
- 支援快速切換日期（昨天、今天、明天）
- 零延遲的日期切換體驗

### 📝 任務管理
- 發布每日任務（作業、訂正、攜帶物品等）
- 自訂任務類型標籤
- 查看任務完成進度

### 👥 學生狀態追蹤
- 查看每位學生（村民）的任務完成狀態
- 顯示完成率與統計資訊
- 小組分組顯示，支援小組完成率計算

### ⚙️ 村莊設定
- 自訂任務類型標籤
- 新增與刪除標籤
- 視覺化的標籤顏色區分

### 🎨 特色設計
- 動物森友會風格的 UI 設計
- DiceBear 頭像系統（Notionists 風格）
- 響應式設計，支援各種螢幕尺寸
- 靜態高亮慶祝效果（高效能）

## 🛠️ 技術棧

- **前端框架**：React 19.2.0
- **建置工具**：Vite 7.2.4
- **樣式**：Tailwind CSS 4.1.18
- **圖標**：Lucide React
- **日期處理**：date-fns 4.1.0
- **日曆組件**：react-calendar 6.0.0

## 📦 安裝與使用

### 環境需求
- Node.js 18+ 
- npm 或 yarn

### 安裝步驟

1. **克隆專案**
   ```bash
   git clone <repository-url>
   cd purr-purr-town
   ```

2. **安裝依賴**
   ```bash
   npm install
   ```

3. **設定環境變數**
   
   建立 `.env` 檔案，設定 API 網址：
   ```
   VITE_API_URL=https://your-api-url.com
   ```

4. **啟動開發伺服器**
   ```bash
   npm run dev
   ```

5. **建置生產版本**
   ```bash
   npm run build
   ```

## 🎯 核心功能說明

### 本地緩存架構 (v7.0)
- 進入村莊時載入所有歷史資料
- 切換日期時不重新載入，直接從本地資料計算
- 實現零延遲的日期切換體驗

### 樂觀更新機制
- UI 立即回應使用者操作
- API 請求在背景執行
- 提供流暢的操作體驗

### 日期格式統一
- 所有日期統一為 `yyyy-MM-dd` 格式
- 確保資料持久化與讀取正確性
